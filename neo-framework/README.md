NEO-FRAMEWORK
=============

##소개
스프링 프레임워크를 이용한 WEB 프레임워크

##필요 환경
- Java 1.6
- Spring Tool Suite version >= 3.4 (option)
- Maven version >= 3.0
- servlet-api version >= 2.5 	

##프레임워크 구성
- native
	- native-core
	- native-spring
- web
	- web-core
	- web-servlet-cms
	- web-servlet-file

##설치 방법
	STS에서 프로젝트 import
